# Week 1 – Networking Fundamentals & Network Simulation Tools
Network Administration Internship Program – IT-Simplera Institute

## Overview
Week 1 task: learned core networking basics and built a simple LAN topology
using Cisco Packet Tracer and GNS3.

## What I Did
- Learned LAN, IP addressing, subnetting, switches, and ping
- Built a LAN (2 PCs + 1 switch) in both Packet Tracer and GNS3
- Configured static IPs (192.168.1.0/24) and verified connectivity via ping
